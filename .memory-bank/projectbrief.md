## Project Overview
Desenvolvimento de um Data Warehouse com análise de Business Intelligence para dados brasileiros da plataforma Steam. O projeto visa proporcionar insights sobre o mercado de jogos brasileiro através de visualizações de dados e dashboards interativos.

**Disciplina**: Business Intelligence (6º período - Noite)  
**Instituição**: PUC Minas - Campus São Gabriel  
**Professor**: Claudiney Vander Ramos  
**Valor**: 20 pontos

## Core Requirements
- ✅ Integração de dados de 3 fontes diferentes (Steam API, SteamCharts, SteamDB)
- ✅ Criação de Data Warehouse modelo dimensional (tabelas fato e dimensão)
- ✅ Implementação de pipeline ETL completo (Extract, Transform, Load)
- ✅ Desenvolvimento de dashboards em 3 níveis decisórios (operacional, gerencial, estratégico)
- ✅ Aplicação de princípios de Data Storytelling
- ✅ Apresentação final da solução funcionando

## Success Criteria
- [ ] Schema SQL executado e banco criado
- [ ] Pipeline ETL extraindo dados com sucesso
- [ ] Dados consolidados no DW sem inconsistências
- [ ] Dashboards visuais implementados e funcionando
- [ ] Apresentação realizada com todos os membros participando

## Scope

### In Scope
- Extração de dados de jogos brasileiros
- Análise de métricas de jogadores (simultâneos, histórico)
- Histórico de preços e descontos
- Frequência de atualizações
- Geração de indicadores (KPIs)
- Dashboards com gráficos, mapas e relatórios

### Out of Scope
- Previsão de vendas (forecasting)
- Análise de reviews/comentários
- Social media analytics
- Customização em tempo real (real-time streaming)
